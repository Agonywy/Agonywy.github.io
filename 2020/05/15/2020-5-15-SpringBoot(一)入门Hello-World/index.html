一：Spring Boot：J2EE开发的一站式解决方案

SpringBoot是我们学习分布式微服务的基础，对Spring做了进一步的封装，简化了第三方包的引用，通过提供的starter，简化了依赖包的配置。

SpringBoot的优点：

  • SpringBoot他对Spring做了进一步的封装，可以轻松创建独立的Spring应用程序。

  • SpringBoot内嵌Tomcat、Jetty等web容器，不需要打包成WAR文件去Tomcat等服务器上部署

  • SpringBoot提供了一系列的“starter”来简化Maven的配置

  • SpringBoot开箱即用，尽可能自动配置Spring

二：Spring Boot环境准备

  • 环境准备： java version “1.8.0_151” + Apache Maven 3.6.1 + IDEA

  • JDK建议1.8以上，Maven建议3.3以上版本，SpringBoot依赖包 1.5.9

  • Maven设置：找到maven/conf/settings.xml配置文件，在profiles标签中添加

  • IDEA设置：把自己的maven整合进来，负责IDEA会使用自带默认的maven

  • Settings—>搜索maven–>配置maven路劲，setting.xml路径，repo仓库路径

  • 继续配置maven，保证maven插件可以对应用进行打包
